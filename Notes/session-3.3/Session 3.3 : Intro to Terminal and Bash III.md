### Text Processing

- **sed**
  - stream editor
  - `sed 's/<oldValue>/<replacementValue>/g' fileName`
- **awk**
  - `awk -F<delim> '{print $NUM1,$NUM2,...}' fileName`
  - Delimeters can be `" "` or `,`
  - Note that the argument for the delimeter has not space after the delimeter flag, `-F`.
